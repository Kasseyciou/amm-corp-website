# AMM website prototype

Static five-page English prototype for Advanced Medical Material Corp.

## Pages

- `index.html` — Home
- `about.html` — About
- `services.html` — Pre-License, Product Launch, and Post-Marketing services
- `regulatory.html` — TFDA QSD review and product registration
- `products.html` — Distributed Products

## Preview locally

From the `site` directory:

```bash
python3 -m http.server 4173
```

Then open `http://127.0.0.1:4173/`.

## Notes

- The supplied horizontal logo is used as the primary brand asset.
- The Home page summarizes the four destination pages and uses project-specific editorial scene images.
- Page copy intentionally uses only the supplied PDF/DOCX content and a compact set of supporting UI labels.
- Product photography comes from images embedded in the supplied DOCX; products without supplied photos use typographic visuals.
- No unsupported market claims, performance statistics, or invented service capabilities are included.
- Market data, product launch years, organization details, GDP certificate details, and case-study statistics require client confirmation before production launch.
